# Counter

Input component for incrementing and decrementing numeric values, plus a badge-style number display.

## Import

```tsx
import { Counter, NumberCounter } from "@raydenui/ui";
```

## Usage

### Basic Counter

<Preview>
  <Counter />
</Preview>

```tsx
<Counter />
```

### Controlled Counter

```tsx
const [value, setValue] = useState(5);

<Counter value={value} onChange={setValue} />
```

### Sizes

<Preview>
  <Counter size="sm" defaultValue={3} />
  <Counter size="md" defaultValue={5} />
  <Counter size="lg" defaultValue={7} />
</Preview>

```tsx
<Counter size="sm" />
<Counter size="md" />
<Counter size="lg" />
```

```tsx

  <ShoppingCartIcon className="w-6 h-6" />

// Notification count

  <span>Messages</span>

</div>
```

## Props

### Counter

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | — | Current value (controlled) |
| `onChange` | `(value: number) => void` | — | Change callback |
| `defaultValue` | `number` | `0` | Initial value (uncontrolled) |
| `size` | `"sm" \| "md" \| "lg"` | `"sm"` | Button and text size |
| `shape` | `"rounded" \| "block" \| "pill"` | `"rounded"` | Container border radius |
| `min` | `number` | `0` | Minimum allowed value |
| `max` | `number` | `Infinity` | Maximum allowed value |
| `step` | `number` | `1` | Increment/decrement amount |
| `disabled` | `boolean` | `false` | Disabled state |
| `className` | `string` | — | Additional CSS classes |

### NumberCounter

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number \| string` | — | Display value (required) |
| `size` | `"sm" \| "md" \| "lg"` | `"sm"` | Badge size |
| `color` | `"orange" \| "red" \| "grey" \| "white"` | `"orange"` | Background color |
| `className` | `string` | — | Additional CSS classes |

## Accessibility

- Increment button has `aria-label="Increase"`
- Decrement button has `aria-label="Decrease"`
- Buttons are disabled when at min/max limits
- Supports keyboard navigation
