# Divider

Dividers separate content into clear groups with optional labels, icons, or buttons.

## Import

```tsx
import { Divider } from "@raydenui/ui";
```

## Usage

### Basic

<Preview>
  <div className="w-full">
    <Divider />
  </div>
</Preview>

```tsx
<Divider />
```

### With Label

<Preview>
  <div className="w-full">
    <Divider label="Or continue with" />
  </div>
</Preview>

```tsx
<Divider label="Or continue with" />
```

### With Title

<Preview>
  <div className="w-full">
    <Divider variant="with-title" title="Section Title" />
  </div>
</Preview>

```tsx
<Divider variant="with-title" title="Section Title" />
```

### With Icon

<Preview>
  <div className="w-full">
    <Divider variant="with-icon" icon={<Icon name="stars" />} />
  </div>
</Preview>

```tsx
import { Icon } from "@raydenui/ui";

<Divider variant="with-icon" icon={<Icon name="stars" />} />
```

### With Button

<Preview>
  <div className="w-full">
    <Divider
      variant="with-button"
      buttonText="Load More"
    />
  </div>
</Preview>

```tsx
<Divider
  variant="with-button"
  buttonText="Load More"
  onButtonClick={() => loadMore()}
/>
```

### Vertical Orientation

  <div className="flex items-center h-8 gap-4">

```tsx
<div className="flex items-center h-8 gap-4">

```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "with-label" \| "with-icon" \| "with-title" \| "with-button"` | `"default"` | Divider variant |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Orientation |
| `label` | `string` | — | Label text (with-label variant) |
| `title` | `string` | — | Title text (with-title variant) |
| `icon` | `ReactNode` | — | Icon (with-icon variant) |
| `buttonText` | `string` | — | Button text (with-button variant) |
| `onButtonClick` | `() => void` | — | Button click handler |
| `className` | `string` | — | Additional CSS classes |

## Accessibility

- Uses `role="separator"` for semantic meaning
- Orientation communicated via `aria-orientation`
- Button variant is keyboard accessible
